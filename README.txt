------------------------------------
КАК ЗАПУСТИТЬ
------------------------------------

1. Запустите основной файл:
   python run_all.py

2. В консоли появится меню выбора спутников.
   - Сначала отобразятся все спутники GPS (G) и ГЛОНАСС (R);
   - Введите номер НКА, чтобы выбрать спутник;
   - Затем выберите эпоху (время измерений).

3. После выбора программа выведет список всех параметров блока эфемерид
   с их значениями и пояснениями.
